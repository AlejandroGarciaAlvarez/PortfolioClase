# Portfolio – Experiencias Inmersivas

## Descripción
Portfolio personal enfocado en experiencias web inmersivas, con animaciones fluidas, navegación clara y enfoque mobile-first. Incluye secciones de presentación, stack tecnológico, proyectos destacados y contacto.

## Demo
- Live: https://tu-demo-aqui.com (reemplaza con tu URL)

## Tecnologías usadas
- HTML5 + Tailwind CSS (CDN) para maquetado y estilos.
- JavaScript (Vanilla) para interacción, menú móvil y toggles de tema/idioma.
- GSAP + ScrollTrigger para animaciones y reveal en scroll.
- Lenis para scroll suave.
- Canvas 2D para fondo de partículas.

## Características
- Diseño responsive con enfoque mobile-first.
- Menú hamburguesa a pantalla completa en móvil con bloqueo de scroll.
- Toggling de tema (día/noche) e idioma (ES/EN).
- Animaciones de entrada y tilt en tarjetas de proyectos.

## Cómo ejecutar
1) Clona el repositorio y abre `index.html` en tu navegador, o sirve la carpeta con un servidor estático.
2) Asegúrate de contar con conexión a Internet para cargar Tailwind y fuentes desde CDN.
